### высоконагруженная база данных MySql, критичная к отказу
Для данной задачи подойдет платформа виртуализации, поддерживающая фермы высокой доступности, например VMWare vSphere. Так же VMWare дает широкие возможности работы с устройствами ввода-вывода.
Задача не вполне четко сформулирована. Для данной задачи можно было бы рассмотреть и физические сервера, так как это исключило бы точку отказа в виде гипервизора виртуализации и обеспечило бы самую высокую производительность за счет исключения оверхеда на виртуализацию.
